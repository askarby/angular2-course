Denne tekst er interpoleret med variablen value som har værdien "{{value}}" (dette er interpolering).

Vi kan også binde variablen til egenskaben value på nedenstående input-felt.

Vi kan tilmed også binde events (hændelser) til at opdatere vores variabel. Nedenstående input-felts keyup-event er blevet bundet til at opdatere indholdet af variablen.
